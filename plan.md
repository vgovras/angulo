## 1. Огляд ринку та конкурентний аналіз

Цефалометричний аналіз (розшифровка ТРГ) — це вимірювання кутів і відстаней між анатомічними точками на рентгенівському знімку черепа. Лікарі-ортодонти та щелепно-лицьові хірурги використовують його для планування лікування. Нижче наведено огляд існуючих рішень, знайдених у відкритих джерелах.

### 1.5 Що НЕ роблять конкуренти (наша ніша)

Жоден з існуючих **безкоштовних** інструментів не вирішує одночасно:

- Ручне калібрування масштабу по лінійці на фото (pixel-to-mm прив'язка)
- Зручне виставлення довільних точок на мобільному без AI
- Автоматичний розрахунок кута між будь-якими 3 точками
- Накладення сітки в реальних сантиметрах поверх знімку
- Повністю офлайн, без реєстрації, без оплати

## 2. Ціль продукту

Lightweight веб-додаток (mobile-first PWA) для **ручного цефалометричного аналізу ТРГ** без реєстрації, без оплати, без AI — лише зручний інструмент для лікарів та студентів, які хочуть швидко виставити точки і отримати кути.

**Цільова аудиторія:**
- Ортодонти та щелепно-лицьові хірурги, які хочуть швидко перевірити кут без спеціального ПЗ
- Студенти-стоматологи, що вивчають цефалометрію
- Пацієнти та їхні родичі, які хочуть самостійно розібратися в своєму знімку

**Ключовий сценарій використання:**
1. Лікар фотографує ТРГ телефоном або отримує цифровий файл
2. Завантажує фото у додаток
3. Калібрує масштаб по лінійці на знімку
4. Виставляє 3+ точки на анатомічних орієнтирах
5. Отримує значення кута в градусах між точками

## 6. UX/UI специфікація для мобільного

### 6.1 Drag handle — детальна специфікація

**Проблема:** коли користувач тягне точку пальцем, він закриває долонею саме місце куди ставить точку. Це відома проблема медичних мобільних застосунків.

**Рішення:**

```
[Canvas]
  ●  ← точка (коло r=6px) — ТОЧНЕ місце
  ╎
  ■  ← handle (44×44px), зміщений на (+20px, -20px)
       (правіше і вище від точки)
```

- Точка на Canvas — маленьке коло (радіус 6px), показує ТОЧНЕ місце
- Handle — квадрат 44×44px, зміщений на `(+20px, −20px)` від центру точки
- Від handle до точки — тонка пунктирна лінія (зв'язок очевидний)
- Handle видно тільки коли точка обрана (`selected`)
- Drag відбувається за handle, але точка переміщується з урахуванням offset

> Handle — кастомний Canvas-елемент, **не** shadcn-svelte компонент. Малюється через `ctx.fillRect()`.

### 6.2 Режими інструменту

Реалізовані через **`Toggle Group`** (shadcn-svelte) у bottom toolbar.

| Режим | Іконка | Поведінка тачу |
|-------|--------|----------------|
| Рухати (Pan) | 🖐️ | Один палець — рухає зображення. Два пальці — zoom |
| Додати точку | 📍 | Тап — додає нову точку. Тап на існуючу — обирає її (показує handle) |
| Калібрувати | 📏 | Тап 1 — перша точка. Тап 2 — друга. Після — `Input` для відстані у `Sheet` |

### 6.3 Bottom Sheet (мобільний drawer)

Реалізований через **`Sheet`** (shadcn-svelte) з `side="bottom"`.

- Відкривається свайпом вгору або кнопкою ☰ (`Button` variant ghost)
- Всередині — **`Scroll Area`** з двома секціями, розділеними **`Separator`**
- **Секція "Точки":** назва (`Input` inline), координати в мм, кнопка видалення (`Button` variant ghost, icon)
- **Секція "Кути":** назва вимірювання, значення (`Badge`), dropdowns вибору точок (`Select`), кнопка видалення
- Тап на рядок точки — центрує Canvas на ній та обирає її

---

### 6.4 Калібрування — flow

1. Натиснути кнопку **Калібрувати** (`Toggle` в `Toggle Group`)
2. Поставити 2 точки на лінійці тапом на Canvas
3. Автоматично відкривається **`Sheet`** (`side="bottom"`) з:
   - текстом інструкції
   - **`Input`** + **`Label`**: "Відстань між точками (мм)"
   - кнопкою **`Button`** "Підтвердити"
4. Після підтвердження — Sheet закривається, у тулбарі з'являється **`Badge`**: "1 см = Xpx"
5. **`Sonner`** toast: "Масштаб налаштовано ✓"

---

### 6.5 Підтвердження небезпечних дій

Кнопка "Очистити все" відкриває **`Dialog`** (shadcn-svelte):

```
Заголовок:  "Очистити всі точки та кути?"
Текст:      "Це видалить X точок та Y вимірювань. Дію неможливо скасувати."
Кнопки:     [Скасувати]  [Очистити]  (destructive variant)
```
